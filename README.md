# Netflix Data Analysis Project

![Python](https://img.shields.io/badge/Python-3.14-blue)
![Pandas](https://img.shields.io/badge/Library-Pandas-yellow)
![Status](https://img.shields.io/badge/Status-Active-success)

A Python data analysis project that analyzes Netflix movies and TV shows using a real CSV dataset.

---

## Features

### 1. Load Dataset

- Read Netflix dataset using Pandas
- Display first 5 rows
- Show dataset shape
- Display column names

### 2. Content Type Analysis

- Count Movies
- Count TV Shows
- Analyze content distribution

### 3. Genre Analysis

- Display top genres
- Analyze most common categories

### 4. Release Year Analysis

- Show most frequent release years
- Analyze yearly content trends

### 5. Country Wise Analysis

- Display top countries
- Analyze content production by country

### 6. Search Feature

- Search movies and TV shows by title
- Case-insensitive search support

### 7. Data Cleaning

- Detect missing values
- Analyze incomplete dataset entries

### 8. Data Visualization

- Generate bar charts using Matplotlib
- Visualize:
  - Movies vs TV Shows
  - Top Release Years
